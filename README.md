# Untis+

This is a project to create a new app for the school schedule software "Untis". The app will be called "Untis+" and will offer additional features and improvements over the original Untis app. The goal is to provide a better user experience and more functionality for students, teachers, and parents who use Untis for managing their school schedules.

## Features

- Improved user interface with a modern design
    - Customizable themes and layouts
    - Material design components for a consistent look and feel
- Real-time synchronization with the Untis server
- Notifications for upcoming classes and events
- Support for multiple languages

## Development

The development of Untis+ will be done using Flutter, a popular framework for building cross-platform mobile applications. The project will be open-source, allowing developers from around the world to contribute and help improve the app.

## Contributing

If you would like to contribute to the development of Untis+, please feel free to fork the repository and submit pull requests. We welcome contributions of all kinds, including bug fixes, new features, and improvements to the user interface.
